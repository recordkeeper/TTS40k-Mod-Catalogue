To use the maps included in this collection: Do the following...

1. Subscribe to these mods: https://steamcommunity.com/sharedfiles/filedetails/?id=1717042451

2. Load the BCB Map Base for Kill Team or 40k (use Apocalypse Map Base if playing Apocalypse)

3. Select the proper mat size (6 x 4 for included 40k maps, 4 x 4 for Killzone maps)

4. Go into Save & Load, and Additive Load the map you want to use onto the BCB Map Base mod.
	4-a. If no mat loads, use the white button on the upper right side of the table edge to choose one manually
